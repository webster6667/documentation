# TypeScript

🔹 int - интерфейс
```typescript
interface $name$ {
    $data$
}
```

<br><br>

🔹 exp:int - экспорт интерфейса
```typescript
export interface $name$ {
    $data$
}
```

<br><br>

🔹 tp:fn - экспорт типизированой функции
```typescript
type $name$ = ($props$) => $result$
```

<br><br>

🔹 exp:tp:fn - экспорт типизированой функции
```typescript
export type $name$ = ($props$) => $result$
```
