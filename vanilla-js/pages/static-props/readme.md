# Статический метод или свойство

Это сущность котороя не зависит от состояния класса  
Всегда возвращают одно и тоже значение  
Не наследуется 


Обращения к статическому методу вне класса  

* Используя имя класса - `MyClass.myStaticMethod()`  
* Через конструктор 

```
 const myClass = new MyClass(); 
 myClass.myStaticMethod
```
