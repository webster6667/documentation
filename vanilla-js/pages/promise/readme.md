# Промисы
> Объект, для выстраивания цеопчки ассинхронных событий

🔹 Имеет три состояния  
&emsp;&emsp; 🎯 `pending` - В ожидании результата  
&emsp;&emsp; 🎯 `rejected` - Произошла ошибка или вызвали `reject`  
&emsp;&emsp; 🎯 `fullfill` - Вызвали `resolve`

🔹 Имеет два параметра  
&emsp;&emsp; 🎯 `resolve`
&emsp;&emsp; 👆          

&emsp;&emsp; 🎯 `reject`
